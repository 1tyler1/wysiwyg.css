Hello World

wysiwyg.css file is a simple collection of styles targeted at HTML elements generated from a WYSIWYG editor (like TinyMCE) or Markdown.

Its purpose are both readibility and simplicity.

And as wysiwyg.css is easily customisable, it also supports themes.

How it works

You just add the .wysiwyg CSS class to a container, and it will style all its child elements: 

<div class="wysiwyg">
  <!-- Your generated HTML -->
</div>

// Spacing
$margin: 1.4em

// Typography
$family-primary: -apple-system, "Helvetica", "Arial", sans-serif
$family-monospace: "Inconsolata", monospace

What it styles

Blocks

  • Headings: from <h1> to <h6>
  • Paragraphs <p>
  • Lists: <ol> and <ul>
  • Blockquotes <blockquote>
  • Tables <table> <tr> <th> and <td>
  • Pre <pre>
  • Figures <figure> and <figcaption>

Inline elements

  • Links: <a>
  • Semantic: <strong> and <em>
  • Code: <code> <kbd> and <samp>
  • Abbreviations: <abbr>
  • Notations: <mark> <s> <del> and <ins>
  • Quotes: <q> and <cite>
  • Other: <small> <time> <sub> <sup> <hr>
